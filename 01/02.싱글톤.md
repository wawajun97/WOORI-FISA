## 싱글톤
- 클래스의 객체를 한개만 생성하여 그 객체만 사용하는 디자인 패턴

### 자바로 구현 시

 ```
  class Singleton {

      // 생성자를 private으로 함으로써 다른 클래스에서 생성 방지
      private Singleton() {} 

      // static으로 싱글톤 객체 생성
      // private으로 설정하여 다른 클래스에서 변경 못하도록
      private static Singleton instance = new Singleton(); 

      // 다른 클래스에서 사용 가능하도록 getter 구현
      public static Singleton getInstance() { 
          return instance;
      }
  }

  //다른 클래스에서 사용 시
  Singleton instance = Singleton.getInstance();
 ```
